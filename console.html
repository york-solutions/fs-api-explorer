Pro Tips

      
      
    
        
        
  • Login with a sandbox account. Visit the FamilySearch Developers
        website to learn about how to get one.
    • 
        
        
  • Links in the response body are clickable and will automatically
        populate the URL when clicked.
    • 
        
        
  • XMLHttpRequest hides some information. View the full request and 
        response in the developer console of your browser.
    • 
        
        
  • If you get a 406 Not Acceptable response it means the Accept header
        was set to a value that the current endpoint does not support so try
        changing the Accept header value.
